# 🎬 Movie Search App

Welcome to **Movie Search App** – a simple and interactive web app to search for movies and see their posters along with a brief overview!  

---

## ✨ Features

- Search any movie by name 🔍  
- Get **poster images** 🖼️ instantly  
- See a **short overview** of the movie 📝  
- Responsive and clean UI 💻  
- Built with HTML, CSS, and JavaScript ⚡
